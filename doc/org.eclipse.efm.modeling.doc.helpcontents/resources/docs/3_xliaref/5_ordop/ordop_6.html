The "|xor|" order operator : exclusivity

Considering 2 processes : P and Q, with exclusivity, both processes are evaluated from the initial context, i.e. without taking into consideration the evaluation of the other.

  • if P succeeds :
    • if Q succeeds : no transitions are taken into account, the systems stays in the initial context
    • if Q fails : the context changes to the context resulting from the execution of P alone
  • if P fails :
    • if Q succeeds : the context changes to the context resulting from the execution of Q alone
    • if Q fails : no transitions are taken into account, the systems stays in the initial context
